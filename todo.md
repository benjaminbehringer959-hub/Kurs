# Video Produktionsplan

## 1. Skizze / Konzept
- [ ] Ideenfindung und Brainstorming
- [ ] Zielgruppe definieren
- [ ] Kernbotschaft festlegen
- [ ] Storyline/Drehbuch erstellen
- [ ] Visuellen Stil/Moodboard entwickeln
- [ ] Benötigte Assets (Bilder, Grafiken, Overlays) identifizieren

## 2. Audio
- [ ] Skript für Voiceover/Kommentar erstellen
- [ ] Sprecher/in auswählen und aufnehmen
- [ ] Hintergrundmusik auswählen
- [ ] Soundeffekte identifizieren und integrieren
- [ ] Audiobearbeitung (Schnitt, Mixing, Mastering)

## 3. Video
- [ ] Szenen planen und aufnehmen (falls eigenes Material)
- [ ] Stock-Footage/Bilder beschaffen
- [ ] Animationen/Grafiken erstellen
- [ ] Videoschnitt (Rohschnitt, Feinschnitt)
- [ ] Text-Overlays/Untertitel hinzufügen
- [ ] Farbkorrektur und Grading
- [ ] Visuelle Effekte hinzufügen

## 4. Ablage / Veröffentlichung
- [ ] Video exportieren (verschiedene Formate/Qualitäten je nach Plattform)
- [ ] Metadaten (Titel, Beschreibung, Tags) erstellen
- [ ] Thumbnail/Vorschaubild gestalten
- [ ] Video auf relevanten Plattformen hochladen (YouTube, Vimeo, Social Media)
- [ ] Backup der Projektdateien erstellen
- [ ] Performance verfolgen und analysieren
